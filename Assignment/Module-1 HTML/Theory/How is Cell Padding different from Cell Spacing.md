##How is Cell Padding different from Cell Spacing? 

These attributes are used to control the layout and "breathing room" within an HTML table.

Cell Padding: This refers to the space inside the cell, between the cell border and the actual content (text or image) inside it. Increasing padding makes the cell look larger and prevents text from touching the edges.

Cell Spacing: This refers to the space between individual cells. It controls the gap between the borders of adjacent cells. If you set cell spacing to zero, the borders of the cells will touch each other.

Example:
```
Cell Padding: <table cellpadding="10"> (Adds 10 pixels of space inside every cell around the text).

Cell Spacing: <table cellspacing="5"> (Creates a 5-pixel gap between each box/cell in the table).
```
