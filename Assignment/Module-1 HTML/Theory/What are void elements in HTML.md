##What are Void Elements in HTML?

 Void elements (also called "empty elements") are unique HTML tags that do not contain any text or other elements inside them. Because they have no content, they do not require a closing tag (</tag>). They are used to insert something specific into the page structure, like a line break, a horizontal line, or a piece of metadata.

Example:

```
<br>: Used to force a line break in a paragraph.

<hr>: Used to create a horizontal thematic break (a line) across the page.

<input type="text">: Used to create a data entry field for a user.

<link rel="stylesheet" href="style.css">: Used to connect a CSS file to the HTML.
```
