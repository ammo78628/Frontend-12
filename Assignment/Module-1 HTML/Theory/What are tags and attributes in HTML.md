##What are Tags and Attributes in HTML?

Tags are the primary building blocks of HTML. They tell the browser how to structure or format the content (e.g., "this is a heading," "this is a link"). Most tags come in pairs: an opening tag <tag> and a closing tag </tag>.

Attributes are modifiers used inside the opening tag to provide additional information or instructions about that tag. They define properties like the source of an image, the destination of a link, or the style of a container. Attributes usually follow a name="value" format.


Example:
```
Tag: <img>

Attributes: src="logo.png" (defines the file path) and alt="Company Logo" (defines the description).

Full Code: <img src="logo.png" alt="Company Logo">
```
